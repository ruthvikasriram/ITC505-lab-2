PROJECTS

DETECTION OF ANIMALS USING CONVOLUTIONARY NEURAL NETWORKS
Vehicle animal collision is most common incident which causes loss to wild life. This intelligent system helps in reducing the roadkill during the active nocturnal hours. The system uses thermal image processing using the camera mount on the vehicle. It extracts the features of the object in front of the vehicle to detect the animal. The main method used to extract the features of the animals is VGG-16. The features are detected by normalizing the radiometric image and then processed by finding the magnitude and gradient of a pixel. The extracted features are given as an input to the basic deep learning model, a one-dimensional convolutional neural network, where binary cross-entropy is used to detect the existence of the object. Thus the animal can be detected.

OPTIMIZED PERFORMANCE MEDICAL IMAGE CLASSIFICATION WITH DEEP TREE CONVOLUTIONAL NEURAL NETWORKS
The performance of two popular convolutional neural network (CNN) architectures, Xception-Net and VGG16, for medical image classification tasks. Specifically, we evaluate their performance on two different datasets: the ISIC dataset for skin lesion classification and the CMNIST dataset for histology image classification. For the ISIC dataset, we use Xception-Net to classify different stages of categories. We fine-tune the pre-trained network on a subset of the ISIC dataset and achieve an accuracy of 91.96%. Our results show that Xception-Net is a powerful deep learning model for skin lesion classification and has the potential to be used in clinical settings for aiding dermatologists in the diagnosis of skin cancer. For the CMNIST dataset, we use VGG16 to classify histology images to classify different stages of categories. We fine-tune the pre-trained network on a subset of the CMNIST dataset and achieve an accuracy of 99.76%.

DESCRIPTION OF TAGS USED ABOVE
  • !DOCTYPE html: Defines the document type and version of HTML being used.
  • head: Contains metadata about the HTML document, such as the title, character set, and links to external resources.
  • footer: Defines a footer section placed at the bottom of a document or section, containing information such as copyright notices, contact information.
  • p: Defines a paragraph.
  • span: Defines a section of content within a paragraph or other block-level element, used here to dynamically display the last modified date.
  • script: Defines client-side JavaScript code to be executed within the HTML document.
  • center: used to center-align content horizontally.
  • h1: Defines the main heading of the document or section (the largest and most important heading).
  • b: Defines bold text.
  • hr: Defines a horizontal rule which is used to separate content sections.
  • ul: Defines an unordered list, where each list item is marked with a bullet point.
  • li: Defines a list item within an ordered or unordered list.
  • br: Defines a line break to create vertical spacing between elements.
  • body: Defines the main content of the HTML document, including text, images, links, and other elements.
